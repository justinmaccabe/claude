Signal Logic
Trend

Carry
Monthly
ETF vs benchmark return differential on DBV, PDBC, and HYG.

Quick Start
01
Create virtual environment
python -m venv venv && source venv/bin/activate
02
Install dependencies
pip install -r requirements.txt
03
Initialise database (downloads 3yr price history)
python -m data.fetcher
04
Run weekly signal job
python -m jobs.weekly --portfolio-value 100000
05
Launch dashboard
streamlit run monitor/dashboard.py
